# Trend Score v1 제품 명세

## 1. 문서 상태

- 버전: `trend-score-v1.0.0`
- 상태: MVP 채택
- 기준일: 2026-07-10
- 적용 범위: 일봉 종가 기반 롱온리 주식·주식형 ETF 후보 스크리너

이 점수는 미래 상승 확률이나 매수 추천이 아니다. 사전에 정의한 상승 추세 조건을 현재 종목이 얼마나 충족하는지를 0~100점으로 표현한다.

## 2. 제품 역할

Trend Score v1의 역할은 다음 세 가지다.

1. 추세 조건에 맞는 종목과 ETF를 찾는다.
2. 선정 또는 제외된 이유를 초보자가 이해할 수 있는 문장으로 설명한다.
3. 후보를 모의투자 전략으로 넘길 수 있게 한다.

Trend Score v1은 다음을 결정하지 않는다.

- 실제 매수 여부
- 보유 종목 수
- 종목별 투자 금액
- 매도 시점
- 현금 보유 비중

이 항목들은 별도의 포트폴리오·모의투자 규칙에서 정의한다.

## 3. 지원 범위

### 비교군

상대강도, 변동성, 거래활성도는 서로 다른 시장과 상품을 섞지 않고 아래 비교군 안에서 계산한다.

| 코드 | 비교 대상 |
| --- | --- |
| `US_STOCK` | 미국 거래소 상장 일반 보통주 |
| `KR_KOSPI` | KOSPI 일반 보통주 |
| `KR_KOSDAQ` | KOSDAQ 일반 보통주 |
| `US_EQUITY_ETF` | 미국 상장 롱온리 주식형 ETF |
| `KR_DOMESTIC_EQUITY_ETF` | 한국 상장 국내 주식형 ETF |
| `KR_OVERSEAS_EQUITY_ETF` | 한국 상장 해외 주식형 ETF |

한 비교군의 점수는 다른 비교군의 점수와 기대수익률 또는 위험을 직접 비교하는 용도로 사용하지 않는다.

### MVP 제외 대상

- 레버리지·인버스 ETF
- ETN, 워런트, 우선주, SPAC
- 채권형·원자재형·파생형 ETF
- 거래정지 또는 현재 거래 불가 종목
- 253개 미만의 유효 일봉을 가진 종목
- 상품 유형이나 기업행사 데이터가 불명확한 종목

## 4. 처리 순서

```text
데이터 적격성 검사
  -> 절대 추세 필수 조건
  -> Trend Score 0~100 계산
  -> 후보 상태 결정
  -> 시장·상품·체결 위험 경고 생성
  -> 선정 이유와 주의 문장 생성
```

필터, 점수, 경고를 분리한다.

- 필터: 후보로 다룰 수 있는가?
- 점수: 추세 원칙을 얼마나 충족하는가?
- 경고: 점수만으로 표현하면 안 되는 위험이 있는가?

## 5. 입력 데이터와 기호

| 기호 | 정의 |
| --- | --- |
| `P` | 현재 수정주가 |
| `SMA50` | 수정주가 50거래일 단순이동평균 |
| `SMA200` | 수정주가 200거래일 단순이동평균 |
| `R63` | `P / P[t-63] - 1` |
| `R126` | `P / P[t-126] - 1` |
| `R12_1` | `P[t-21] / P[t-252] - 1` |
| `H252` | 최근 252거래일 수정주가 최고값 |
| `VOL60` | 최근 60일 로그수익률 표준편차 × `sqrt(252)` |
| `ADV60` | 최근 60일 `실제 종가 × 거래량`의 중앙값 |
| `PctRank` | 같은 날짜·비교군 안의 백분위 순위, 0~1 |

가격 추세와 수익률은 배당·분할을 반영한 수정주가로 계산한다. 거래대금과 모의 체결가는 당시 실제 가격과 거래량으로 계산한다.

## 6. 데이터 적격성

아래 조건을 모두 만족해야 점수를 정상 산출한다.

- 유효 일봉 253개 이상
- 최근 60거래일 중 유효 거래일 55일 이상
- 수정주가·실제 가격·거래량 데이터가 유효함
- 지원 상품 유형에 해당함
- 현재 거래정지 상태가 아님
- 비교군을 결정할 수 있음
- 비교군 종목 수가 30개 이상임

비교군 종목 수가 30개 미만이면 정의된 상위 비교군으로 한 번 확장한다. 확장 후에도 30개 미만이면 상대순위를 중립값으로 임의 대체하지 않고 `INSUFFICIENT_PEERS`로 처리하여 공식 후보에서 제외한다.

## 7. 유동성 처리

유동성은 두 역할로 나눈다.

### 거래활성도 점수

비교군 안에서 `log(1 + ADV60)`의 백분위 순위를 계산해 최대 5점을 부여한다. 화면에서는 이 항목을 `유동성`보다 정확한 `거래활성도`로 표시한다.

### 후보 적격성 필터

아래 조건을 모두 통과해야 한다.

```text
ADV60 >= market_min_adv[peer_group]
planned_order_value / ADV60 <= 0.001
```

- `market_min_adv`는 시장·상품별 설정값이며 데이터 공급자와 모의투자 규모가 정해질 때 별도 버전 설정으로 고정한다.
- `planned_order_value`가 아직 없으면 사용자가 설정한 기본 1회 주문금액을 사용한다.
- 주문금액이 `ADV60`의 0.1%를 넘으면 후보에서 제외한다.
- `market_min_adv`가 설정되지 않은 환경에서는 연구 점수만 계산할 수 있고 공식 후보 목록과 모의주문은 만들지 않는다.

ETF의 스프레드, NAV 괴리율, 추적오차는 Trend Score에 넣지 않는다. 데이터가 있으면 별도 경고로 보여주며, 라이브 거래를 추가하기 전에는 ETF 적격성 필터로 승격해야 한다.

## 8. Trend Score 계산식

```text
Trend Score
= 장기 추세 구조       30점
 + 절대 모멘텀         25점
 + 상대강도            20점
 + 52주 고점 근접도    10점
 + 변동성 안정성       10점
 + 거래활성도           5점
```

최종 점수는 소수점 첫째 자리로 반올림한다.

### 8.1 장기 추세 구조: 30점

```text
15 × I(P > SMA200)
+ 10 × I(SMA50 > SMA200)
+  5 × I(SMA200[t] > SMA200[t-20])
```

`P > SMA50`은 중복 배점을 줄이기 위해 점수에 넣지 않는다. 현재가가 50일선 아래이면 `단기 추세 약화` 경고를 생성한다.

### 8.2 절대 모멘텀: 25점

```text
 5 × I(R63 > 0)
+10 × I(R126 > 0)
+10 × I(R12_1 > 0)
```

이 항목은 상승 폭이 아니라 여러 기간에서 상승 방향이 일관적인지를 평가한다.

### 8.3 상대강도: 20점

```text
relative_momentum = 0.20 × R63 + 0.40 × R126 + 0.40 × R12_1
relative_strength_score = 20 × PctRank(relative_momentum)
```

같은 날짜의 동일 비교군 안에서만 순위를 계산한다. 약세장에도 상대강도 1등은 존재하므로 아래 절대 추세 필수 조건과 함께 사용한다.

### 8.4 52주 고점 근접도: 10점

```text
high_ratio = P / H252
high_proximity_score = 10 × clamp((high_ratio - 0.80) / 0.15, 0, 1)
```

- 52주 고점의 80% 이하: 0점
- 52주 고점의 95% 이상: 10점

### 8.5 변동성 안정성: 10점

```text
volatility_score = 10 × (1 - PctRank(VOL60))
```

상승 가능성을 예측하는 항목이 아니라, 같은 추세 후보라면 상대적으로 안정적인 가격 흐름에 가점을 주는 품질 항목이다.

### 8.6 거래활성도: 5점

```text
activity_score = 5 × PctRank(log(1 + ADV60))
```

점수와 별도로 7절의 실제 후보 적격성 필터를 반드시 적용한다.

## 9. 절대 추세 필수 조건

점수가 높아도 아래 조건을 하나라도 통과하지 못하면 후보에서 제외한다.

```text
data_eligible
AND liquidity_eligible
AND P > SMA200
AND R126 > 0
```

대표 제외 사유 코드는 다음과 같다.

| 코드 | 의미 |
| --- | --- |
| `INSUFFICIENT_HISTORY` | 가격 이력 부족 |
| `MISSING_DATA` | 필수 데이터 누락 |
| `INSUFFICIENT_PEERS` | 비교군 표본 부족 |
| `INSTRUMENT_NOT_SUPPORTED` | 지원하지 않는 상품 |
| `SUSPENDED` | 거래 불가 상태 |
| `BELOW_SMA200` | 현재가가 200일선 이하 |
| `NEGATIVE_6M_MOMENTUM` | 6개월 수익률이 0 이하 |
| `LOW_ABSOLUTE_LIQUIDITY` | 시장별 최소 거래대금 미달 |
| `ORDER_TOO_LARGE` | 계획 주문금액이 ADV60의 0.1% 초과 |

## 10. 후보 상태

| 상태 | 조건 |
| --- | --- |
| `STRONG_CANDIDATE` | 필수 조건 통과 및 80점 이상 |
| `CANDIDATE` | 필수 조건 통과 및 65~79.9점 |
| `WATCH` | 필수 조건 통과 및 50~64.9점 |
| `WEAK` | 필수 조건 통과 및 50점 미만 |
| `EXCLUDED` | 필수 조건 하나 이상 실패 |
| `NOT_AVAILABLE` | 데이터 적격성 실패 |

65점과 80점은 미래 수익률을 보장하는 통계적 경계가 아니라 v1 화면 분류 기준이다. 검증 결과에 따라 바꿀 때는 기존 결과를 덮어쓰지 않고 점수 버전을 올린다.

### 상태 안정화

- 원시 점수: 각 시장 종가 확정 후 매일 갱신
- 공식 후보 목록: 시장별 주간 마지막 거래일 종가 후 갱신
- 신규 후보: 필수 조건 통과 및 65점 이상
- 기존 후보 유지: 필수 조건 통과 및 60점 이상
- 후보 해제: 필수 조건 실패 또는 60점 미만

이 65/60 차이는 수익률을 높이기 위한 규칙이 아니라 횡보장에서 후보 상태가 반복 변경되는 것을 줄이는 UI·운영 규칙이다.

## 11. 점수와 분리할 경고

경고는 Trend Score를 변경하지 않는다.

| 경고 코드 | 초기 조건 | 초보자용 의미 |
| --- | --- | --- |
| `WEAK_MARKET` | 대표지수 `<= SMA200` | 종목은 강하지만 시장 전체 흐름은 약함 |
| `BELOW_SMA50` | `P <= SMA50` | 장기 추세는 유지되지만 최근 흐름이 약해짐 |
| `HIGH_VOLATILITY` | `VOL60` 비교군 상위 20% | 같은 유형보다 가격 등락이 큼 |
| `SHORT_TERM_OVERHEAT` | `P / SMA50 - 1` 비교군 상위 5% | 단기 급등으로 50일선과 거리가 큼 |
| `WIDE_ETF_SPREAD` | 스프레드 데이터 기준 초과 | 시장가 주문 시 불리한 가격에 체결될 수 있음 |
| `ETF_NAV_GAP` | NAV 괴리 기준 초과 | 시장가격과 기초가치의 차이가 큼 |
| `FX_EXPOSURE` | 환노출 미국 종목·해외 ETF | 원화 수익률이 환율의 영향을 받음 |
| `STALE_DATA` | 시장 마감 후 데이터 지연 | 최신 종가가 반영되지 않았을 수 있음 |

미국 종목 점수는 USD 가격으로 계산하고 원화 환산 수익률과 환율 기여도는 별도로 표시한다. 한국 상장 환노출 해외 ETF는 이미 환율이 가격에 반영되므로 이중 환산하지 않는다.

## 12. 초보자용 화면 규칙

첫 화면의 후보 카드는 아래 정보만 우선 노출한다.

```text
[추세 점수 84점] [강한 추세 후보]
미국 주식 비교군 상위 15% · 2026-07-09 종가 기준

선정 이유 최대 3개
주의사항 최대 2개
후보에서 제외되는 조건
```

세부 점수 6개와 계산값은 `자세히 보기`에서 제공한다.

화면 문구에는 `매수`, `추천`, `상승 확률`을 사용하지 않는다. 다음 문구를 항상 표시한다.

> 추세 점수는 사전에 정의한 가격 추세 조건의 충족도를 나타냅니다. 미래 수익률이나 상승 가능성을 보장하지 않습니다.

설명은 생성형 AI가 자유롭게 작성하지 않고 계산 결과와 이유 코드에 연결된 결정적 템플릿으로 생성한다.

설명 예시:

- `현재가는 200일 이동평균선보다 8.4% 높아 장기 상승 추세 조건을 충족합니다.`
- `3개월, 6개월, 12-1개월 수익률이 모두 양수입니다.`
- `복합 모멘텀은 미국 주식 비교군 상위 12%입니다.`
- `최근 60일 변동성이 비교군 상위 8%로 가격 등락이 큰 편입니다.`
- `현재가가 200일선 아래에 있어 후보에서 제외됐습니다.`

## 13. 백테스트 규칙

### 신호와 체결

```text
t일 시장 종가 확정
-> t일 점수와 후보 상태 계산
-> 해당 시장 캘린더의 다음 거래 가능 시점에 모의 체결
```

다음 거래일은 주간 데이터의 다음 행을 `shift(-1)`로 구하지 않는다. 거래소별 캘린더에서 다음 실제 거래일을 찾고, 거래정지·거래량 0·가격제한 상태라면 체결을 미루거나 미체결로 처리한다.

### 필수 원칙

- 해당 날짜에 실제 존재했던 종목만 비교군에 포함
- 상장폐지·합병된 종목과 회수 가능 금액 반영
- 당시 알 수 있었던 데이터만 사용
- 신호는 수정주가, 체결은 실제 가격 사용
- 수수료, 세금, 스프레드, 슬리피지 반영
- 거래정지 기간을 선형 보간하지 않음
- 미국·한국 거래일과 마감 시각을 별도 처리
- 전략 버전, 데이터 스냅샷, 설정값을 결과와 함께 저장

### 스크리너 검증

점수 자체의 품질은 포트폴리오 성과와 분리해 평가한다.

- 점수 구간별 1·3·6·12개월 후 수익률
- 점수 구간별 수익률의 단조성
- 점수와 미래 수익률의 Spearman 순위상관
- 선정 후 최대 불리 움직임과 최대 유리 움직임
- 후보 상태 유지 기간과 교체율
- 시장·상품·시기별 결과 안정성
- 각 구성요소 제거 실험

포트폴리오 백테스트는 보유 종목 수, 비중, 리밸런싱, 해제 조건, 현금 규칙을 별도 명세한 뒤 실행한다.

## 14. 검증 전 고정한 가설

다음 값은 학술적으로 정해진 정답이 아니라 설명 가능하고 검증하기 쉬운 v1 제품 가설이다.

- `30/25/20/10/10/5` 가중치
- 65점 후보, 80점 강한 후보 경계
- 65점 진입, 60점 유지 규칙
- 3·6·12-1개월 모멘텀 배합
- 거래금액이 ADV60의 0.1% 이하라는 주문 규모 제한

백테스트 결과가 좋다는 이유로 여러 값을 반복 변경해 최고 성과 조합을 고르지 않는다. 넓은 범위의 민감도와 표본외 결과를 확인하고, 변경이 필요하면 `trend-score-v2`로 새 버전을 만든다.

## 15. 구현 완료 조건

- 동일 입력과 설정에서 항상 동일 점수와 이유가 생성됨
- 여섯 구성점수의 합이 최종 점수와 일치함
- 데이터 부적격 종목에는 정상 점수가 생성되지 않음
- 필수 조건 실패 시 점수와 별개로 제외 사유가 표시됨
- 비교군 간 백분위가 섞이지 않음
- 경고가 점수 값을 변경하지 않음
- 종가 신호가 같은 날 종가로 체결되지 않음
- 모든 결과에 점수 버전과 기준 시각이 저장됨
- 경계값과 결측값에 대한 자동 테스트가 존재함

## 16. 알려진 한계

- 실적 발표 후 갭 하락, 거래정지, 하한가 같은 급격한 사건을 미리 방지하지 못한다.
- 약세장 급반등 구간에는 기존 모멘텀 순위가 빠르게 뒤집힐 수 있다.
- 높은 점수는 기대수익률이나 손실 가능성을 직접 나타내지 않는다.
- 일봉 OHLCV만으로 ETF의 실제 체결 품질을 완전히 판단할 수 없다.
- 이 스크리너만으로는 완성된 투자 전략이나 모의 포트폴리오가 되지 않는다.

## 17. 주요 연구 근거

- [Jegadeesh & Titman, Returns to Buying Winners and Selling Losers](https://doi.org/10.1111/j.1540-6261.1993.tb04702.x): 개별주 중기 상대 모멘텀의 대표적 근거
- [Kenneth French Data Library, Daily Momentum Factor](https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/Data_Library/det_mom_factor_daily.html): 최근 1개월을 제외한 과거 2~12개월 수익률 구성 사례
- [Moskowitz, Ooi & Pedersen, Time Series Momentum](https://w4.stern.nyu.edu/facdir/lpederse/papers/TimeSeriesMomentum.pdf): 여러 자산군의 1~12개월 시계열 모멘텀 연구
- [George & Hwang, The 52-Week High and Momentum Investing](https://doi.org/10.1111/j.1540-6261.2004.00695.x): 52주 고점 근접도와 모멘텀의 관계

이 연구들은 Trend Score v1의 구성요소에 방향성을 제공하지만, 롱온리 미국·한국 주식·ETF 스크리너의 성과나 `30/25/20/10/10/5` 배점을 직접 검증하지 않는다. 배점과 후보 경계는 반드시 별도의 표본외 검증을 거쳐야 하는 제품 가설이다.
